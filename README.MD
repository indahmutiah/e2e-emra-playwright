# e2e-Emra-bootcamp

This is a sample project for e2e testing with sauce labs. Just for learning purpose to understand the automation process. 


## Installation

```bash
npm install
```

## Run

```bash
npm test
```

## Run UI

```bash
npm test:ui
```

## Run headed

```bash
npm test:headed
```

## Run debug

```bash
npm test:debug
